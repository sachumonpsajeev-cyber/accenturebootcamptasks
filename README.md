## 🤖 Featured Project — AI Chatbot with Cognigy.AI

> Built a fully functional AI conversational chatbot using Cognigy.AI 
> platform during the bootcamp.

### What is Cognigy.AI?
Cognigy.AI is an enterprise-grade conversational AI platform used to build 
intelligent chatbots and virtual agents for customer service and automation.

### What I Built
- Designed and deployed a conversational AI chatbot flow
- Configured intents, entities and NLP settings
- Built multi-turn conversation logic
- Integrated chatbot with external data sources
- Tested and optimised bot responses

### Key Concepts Covered
| Concept | Detail |
|---|---|
| **Intent Recognition** | Training the bot to understand user goals |
| **Entity Extraction** | Pulling key data from user messages |
| **Flow Design** | Building conversation logic and branching |
| **NLP Configuration** | Natural Language Processing settings |
| **Deployment** | Publishing and testing the live chatbot |
